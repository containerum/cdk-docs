#### <a name="logs">logs</a>

**Description**:

View pod logs. Aliases: log

**Example**:

logs pod_label [container] [--follow] [--prev] [--tail n] [--quiet]

**Flags**:

| Short | Name | Usage | Default value |
| ----- | ---- | ----- | ------------- |
| -f | --follow | follow pod logs | false |
| -t | --tail | print last <value> log lines | 100 |
